# 8大题材套路速查

## 1. 史诗动作/战斗
- **结构**：wide establishing → close combat → slow-mo climax → wide pull-back
- **光线**：golden hour、moonlit silver、dramatic backlighting
- **关键词**：impact, shockwave, debris, explosion, collision
- **SFX描述是加分项**，典型长度 300-600词
- **推荐模板**：A（电影级分镜）或 D（Beat-Synced）

## 2. FPV 飞行穿越
- **结构**：一段话，逗号堆叠视觉元素
- **光线**：golden sunrise、volumetric lighting
- **关键词**：soaring, diving, weaving, rushing, gliding
- **越简洁越好**，50-150词
- **推荐模板**：C（FPV穿越）

## 3. 超现实/变形
- **核心**：transformation逻辑、过渡描述、材质变化的物理描写
- **光线**：bioluminescent、ethereal glow
- **关键词**：morphs, dissolves, materializes, shatters, reforms
- **推荐模板**：E（一镜到底）

## 4. 动画/卡通
- **核心**：style声明（Pixar, Ghibli, chibi）、表情描写、喜剧节奏
- **对话和音效描述提升效果**
- **关键词**：kawaii, exaggerated, bouncy, expressive
- **推荐模板**：A 或 F

## 5. 人像/变装/写真
- **核心**：Identity Lock（@Image）、光线设计（逆光/侧光/伦勃朗光）、镜头焦段
- **变装类**：每1-2秒一个造型 + 对应转场动作（转身/甩发/扇遮面）
- **关键词**：50mm人像焦段、f/2.8、浅景深、胶片颗粒、丝滑过渡、卡点
- **推荐模板**：F（中文分镜）或 D（Beat-Synced卡点）

## 6. 产品/广告
- **核心**：产品名称明确、使用场景、色彩和光线精确
- **JSON结构化效果最佳**
- **关键词**：showcase, highlight, reveal, premium, 360° orbit
- **推荐模板**：B（JSON结构化）

## 7. 城市/科技
- **核心**：FPV穿越运镜、快速闪剪、场景空间折叠转换
- **关键词**：赛博朋克、霓虹、全息投影、极速俯冲、Hyperlapse
- **推荐模板**：C（FPV穿越）或 E（一镜到底）

## 8. 幽默/创意
- **核心**：反差设定 + 荒诞细节，用一句话传达核心创意
- **越短越好**，创意本身才是关键
- **关键词**：荒诞、无厘头、反差、一本正经
- **推荐模板**：任意，但提示词宜短不宜长
