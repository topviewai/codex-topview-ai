# 高频术语速查表

## 镜头景别
| 缩写 | 英文 | 中文 |
|------|------|------|
| ECU | Extreme Close-Up | 极端特写（眼睛/手指） |
| CU | Close-Up | 特写（脸部） |
| MCU | Medium Close-Up | 中近景 |
| MS | Medium Shot | 中景（腰部以上） |
| WS | Wide Shot | 全景/远景 |
| EWS | Extreme Wide Shot | 超远景 |
| OTS | Over-The-Shoulder | 过肩镜头 |
| POV | Point of View | 主观视角 |
| — | Bird's-eye | 俯瞰 |
